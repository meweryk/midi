Аудио Фазовращатель: принцип работы и настройка

О фазовом вращении

Фазовое вращение на 90° между каналами - мощный аудиоэффект, создающий объёмный звук и глубину. Метод основан на преобразовании, сдвигающем фазу сигнала при сохранении амплитуды.

В акустике фазовый сдвиг создаёт "вихревые резонансы", когда звук кажется исходящим из разных точек пространства. Это происходит за счёт интерференционных картин, усиливающих или ослабляющих определённые частоты.

Настройка параметров

Мин. частота 100 Гц
Нижняя граница частотного диапазона. Для басов: 50-150 Гц
Макс. частота 10000 Гц
Верхняя граница диапазона. Для высоких частот: 8000-12000 Гц
Фильтры 9
Количество фильтров (5-15). Больше = точнее сдвиг, но выше задержка
Добротность 1.0
Регулирует крутизну среза фильтров (0.8-1.2 оптимально)
Задержка 8 мс
Компенсация рассинхронизации каналов. Тестируйте с шагом 1 мс

Визуализация сигнала

Сэмплы 512
Отображаемое временное окно. Для НЧ: 1024, для ВЧ: 256
Амплитуда 0.8
Масштаб сигнала на графике. Уменьшайте при перекрытии
Толщина 2
Толщина линий графика. Для деталей: 1, для чёткости: 3

Практические рекомендации

Точность сдвига:
  • Используйте синус 1 кГц и белый шум для теста
  • Пики одного канала должны совпадать с нулями другого
  • Корректируйте задержку при рассинхронизации
Оптимизация:
  • Для минимальной задержки: 7 фильтров + 1024 сэмпла
  • При искажениях регулируйте добротность (0.7-1.3)
  • Для сложных сигналов: 11 фильтров

Физика вихревых резонансов

Эффект создаёт иллюзию объёмного звучания через имитацию разницы времени прихода звука к ушам (эффект Хааса). При точном 90° сдвиге возникают стоячие волны с круговой поляризацией, вызывающие резонансные явления в помещениях.

Взаимодействие фазово-сдвинутых волн создаёт области усиления/ослабления звука, формируя акустический вихрь. Экспериментируя с настройками, можно управлять плотностью этого вихря - от мягкого вращения до интенсивной турбулентности.
